    About PROFESSOR

    Wooil Kim
    Associate Professor

    Email: wikim@inu.ac.kr

    Research Interests 1. Speech Processing / Recognition - large vocabulary continuous speech recognition system
    - environmental robust processing for ASR, missing-feature theory     2. Human Behavior Sinal Processing - in-vehicle human interface, robust speech recognition for in-vehicle system
    - human emotion processing, signal processing / estimation under cognitive stress / distraction
    3. Information Retrieval System - multimedia / music information retrieval, spoken document retrieval
    - keyword spotting / detection, data mining, dialog system
    4. Statistical Signal Processing, Pattern Recognition, Neural Network
